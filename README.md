#  Python3 SDK for Sinric Pro

### Get your api from [sinric-pro-website](https://sinric.pro)

### Copy paste your credentials into credentials.py

### Required packages
        websockets (python3 -m pip install websockets)

### Run App.py

       python3 app.py
